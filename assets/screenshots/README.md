## Capture tips

- **Use a clean test record** with placeholder data (e.g., "Test User" / "test@your-agency.gov") — never real user data
- **Capture in a 1280-1400px wide browser window** — wide enough to show the full layout, narrow enough to be readable
- **Crop tightly** — remove browser chrome, scroll bars, and empty whitespace
- **PNG format** — better for UI screenshots than JPEG
- **File size** — under 500KB each; use TinyPNG or similar to compress before committing
